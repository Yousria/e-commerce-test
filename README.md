# E-commerce-test-site

## Getting Started

Checkout this repository then execute :

    yarn install
    
it will install all dependencies of the project. Then to start the server : 

    yarn start
    
It will open automatically a browser window : http//localhost:8080

## Development notes

### Architecture overview

The application is packaged using Webpack and is composed of the following main blocks:

    - ReactJS, https://reactjs.org
    - Redux, https://redux.js.org
    - React Material, https://material-ui.com
    - Json-server API, https://github.com/typicode/json-server

### Entry point 

The entry point to the React app is in 

    src/index.js
    
There is no routing.
    
### Redux

The application uses Redux to manage state.
